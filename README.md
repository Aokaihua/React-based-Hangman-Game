# 小游戏简介
<p>
Hangman是一个双人猜单词游戏。一位玩家想一个字，另一位尝试猜该玩家所想的字中的每一个字母。</p>

<p>在本游戏中，浏览器将作为提供单词的玩家，该单词对应的待猜测的字母会以下划线的形式出现。您只需要键入键盘中的任意一个字母，浏览器界面上就会告知您猜测的情况：若目标单词包含您键入的字母，则该字母会出现在对应的下划线上；若不包含，那么您将失去一次猜测的机会。若错误次数累计到6次，则游戏结束。</p>
